# ?? Farm Management Dashboard — .NET MAUI App

A cross-platform **.NET MAUI** application designed to simplify farm monitoring through a clean, responsive dashboard.  
This project highlights **MVVM architecture**, **data binding**, **Shell navigation**, **Microcharts**, and **modern XAML UI design**.

## ?? Features

### ?? MVVM Architecture
- Clean separation of View, ViewModel, and Model.
- Observable properties for real-time UI updates.
- Commands for user interactions and navigation.
- No UI logic in the code-behind.

### ?? Data Binding
- One-way and two-way binding across the app.
- Live metrics for animals, sheep, cows, and profit/loss.
- String formatting for numeric displays.
- Fully reactive UI with minimal boilerplate.

### ??? Shell Navigation
- Centralized routing through `.NET MAUI Shell`.
- Easy, scalable folder-based navigation.
- URI-style navigation calls from ViewModels.
- Clean and readable AppShell structure.

### ?? XAML UI Design
- Fully custom UI created with XAML.
- Responsive layout using Grid, StackLayout, and ScrollView.
- Hero background image with overlay using `AspectFill`.
- Rounded dashboard cards with shadow effects.
- Minimal, consistent farm-themed color palette.

### ?? Microcharts Integration
- Lightweight, cross-platform charting.
- Dashboard analytics and visual data representation.
- Smooth performance on all platforms.

## ??? Tech Stack

| Layer | Technology |
|-------|------------|
| UI | XAML, .NET MAUI Controls |
| Architecture | MVVM |
| Language | C# |
| Navigation | .NET MAUI Shell |
| Charts | Microcharts |
